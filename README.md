# Voice-to-Text Converter Application

## Overview
A real-time desktop application that converts live voice input into text and evaluates spoken mathematical expressions. The project focuses on speech recognition, real-time processing, and responsive user interface design, with support for saving recognized voice history.

## Key Features
- Continuous speech recognition
- Live voice-to-text transcription
- Speech-to-math expression conversion and evaluation
- Dual operating modes: Text and Math
- Timestamped history of recognized voice input
- Option to save voice conversion history to a file
- Real-time UI updates with robust error handling

## Tech Stack
- Python
- Tkinter & ttkbootstrap
- SpeechRecognition
- NumPy
- PyAudio

## Learning Outcome

- Worked with multi-threading to maintain a responsive UI
- Designed history tracking and file export functionality
- Integrated speech recognition with mathematical expression logic
